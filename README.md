# 545 CVs with NER Annotations

*CVs were obtained from open sources and were annotated by ourselves using [**Doccano**](https://github.com/doccano/doccano) Open source annotation tool*

## NER Annotations:

*   Name
*   Email Address
*   Designation
*   Job Specific Skill
*   Tech Tools
*   Companies worked at 
*   Location
*   Years of Experience
*   College Name
*   Degree
*   Graduation Year
*   Soft Skills
